How to download the RRZE Icon Set

If you want to use the RRZE Icon Set, you can stay up-to-date by using the SVN repository. You can check out the repository with the following command:
svn checkout http://svn.berlios.de/svnroot/repos/rrze-icon-set/trunk/tango
To update your local copy, just type

  • svn update or
  • svn up

as a short term.

If you are using Windows, please rely on TortoiseSVN.

If you just want to use the RRZE Icon Set for your own projects at a specific point of time, use the SVN export function:
svn export http://svn.berlios.de/svnroot/repos/rrze-icon-set/trunk/tango.

For details see the Berlios page on SVN.
